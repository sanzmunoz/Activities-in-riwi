# mySQL

### Definition

Base de datos open souce mas popular del mundo.
Sistema de gestion de bases de datos relacionales (RDBMS) de codigo abierto que se utiliza para almacenar y gestionar datos.
SQL : Lenguaje de consulta estructurado
lenguaje de programacion que se usa para recuperar, suprimir, maniplar baases de datos relacionales

entiedades y registros son equivalentes a columnas y filas

-   Crea una base de datos: CREATE DATABASE nombre_db; 
-   Ver tablas creadas: SHOW DATABASES;
-   Posicionarse en la base de datos:  USE nombre_db Database changed;
-   Verificar posicion: SHOW TABLES;
-   Crear tabla en db: CREATE TABLE nombre_tabla (entidad_colum_1 INT AUTO_INCREMENT PROMARY KEY, entidad_colum_2 VARCHAR(20) NOT NULL ...);
-   Insertar datos en tabla: INSERT INTO
