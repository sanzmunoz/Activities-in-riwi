### base de datos no relacional

Sistema de almacenamiento flexible que gestiona datos no estructurados o semiestructurados sin usar tablas fijas ni SQL como lenguaje primario. Diseñadas para gran escalabilidad horizontal, alta velocidad y manejo de grandes volumenes de informacion.

- No requieren estructura predefinida, facilita el manejo de datos cambiantes.
- se escala horizontalmente, añadiendo mas servidores en lugar de potenciar uno solo.
- Almacena datos en formatos JSON o BSON (EJ. MongoDB)
- clave valor: alta velocidad para cache y sesiones
